2. Grundprinzip: Lokale Datenhaltung

SYNR ist eine komplett im Browser laufende Web-Anwendung. Alle Daten (Presets, Einstellungen, erzeugte Sounds) werden ausschliesslich lokal auf dem Geraet des Nutzers gespeichert (localStorage). Es findet kein Cloud-Sync, keine Datenuebermittlung an Server und keine Weitergabe an Dritte statt. Der Anbieter hat keinen Zugriff auf gespeicherte Presets oder sonstige Nutzerdaten.

3. Welche Daten werden verarbeitet?

Ausschliesslich lokal verarbeitet:

  • Synthesizer-Einstellungen und Parameter
  • Gespeicherte Presets
  • Exportierte JSON- und WAV-Dateien

Diese Daten verbleiben im Browser (localStorage) und verlassen das Geraet nicht.

4. Audio-Verarbeitung

Die Klangsynthese erfolgt vollstaendig im Browser mittels der Web Audio API. Es werden keine Audiodaten an externe Server gesendet.

5. Tracking, Analyse & Werbung

SYNR verwendet kein Tracking, keine Analyse-Tools und keine Werbe-SDKs. Kein Firebase, keine Analytics, kein User-Profiling.

6. Export & Import

JSON-Export/Import sowie WAV-Export erfolgen lokal. Dateien werden direkt im Browser erzeugt und auf dem Geraet des Nutzers gespeichert.
